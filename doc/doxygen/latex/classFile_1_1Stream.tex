Deserialises an object from the stream using either internal or external deserialisation. 

This method provides a unified interface for reading objects that may implement deserialisation through either\+:
\begin{DoxyEnumerate}
\item An internal {\ttfamily \doxylink{structFile_1_1Deserialiser}{Deserialiser}} type (Has\+Internal\+Deserialiser\+Impl), or
\item An external {\ttfamily \doxylink{structFile_1_1Deserialiser}{Deserialiser}} specialization (Has\+External\+Deserialiser\+Impl)
\end{DoxyEnumerate}
